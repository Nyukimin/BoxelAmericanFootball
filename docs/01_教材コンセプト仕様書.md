# 01_教材コンセプト仕様書

- 目的: 本作の最上位目的を定義する
- 優先度: 最上位
- 対象: 初心者マネージャー、新入部員、指導者、開発者

## 1. 位置づけ

本作は、初心者マネージャーがアメフトの試合を理解し、チームを支えるための基本視点を身につける教材である。

新入部員への教育にも使えるが、主対象はマネージャーである。マネージャーが試合を見て、記録し、用語を理解し、選手状態に気づき、チームの動きを支えられるようになることを第一目的にする。

ゲーム性は副次的な飾りではない。教材を継続して遊ばせ、理解を定着させるための中核である。ゲーム性を削ぎ落として説明教材に寄せすぎない。

## 2. ひとことで言うコンセプト

マネージャー目線でアメフトの試合を読み解く、ぽっちゃり動物の采配・記録学習ゲーム。

補助表現:

> 可愛い動物チームのそばで、試合の流れ・作戦・記録・用語・選手ケアを少しずつ覚える。遊ぶうちに、マネージャーとして試合が見えるようになる。

## 3. 対象ユーザー

### 3-1. 主対象

- アメフトに初めて触れるマネージャー
- これからチームに関わる学生マネージャー
- 試合撮影、記録、サイドライン補助、用語理解に不安がある人
- ルールブックを読む前に、試合の全体像を体感したい人

### 3-2. 副対象

- 新入部員
- アメフト観戦初心者
- チーム説明会、体験入部、導入研修で使う教材を探している人

### 3-3. 想定しない主対象

- 既にアメフトの戦術を深く理解している上級者
- 選手を直接操作する本格スポーツアクションを求めるユーザー
- NFLレベルの細かいルール再現を期待するユーザー

## 4. 身につける力

本作が育てる力は、操作技術ではなく「試合を見て理解する力」である。

| 分野 | 身につける力 | ゲーム内での対応 |
|---|---|---|
| 試合状況 | ダウン、残りヤード、攻撃権、フィールド位置が分かる | HUD、フィールド表示、コーチの一言 |
| 作戦理解 | ラン、ショートパス、ロングパス、隊形の狙いが分かる | 隊形カード、プレー選択、結果解説 |
| 記録 | プレー結果を短く記録できる | 記録・分析マネージャー、プレー後メモ |
| 用語 | 現場で聞く基本用語に慣れる | 初出解説、用語図鑑 |
| 選手ケア | 選手状態がプレー選択に関わると分かる | テーピング演出、コンディション表示 |
| 観察 | どこを見るべきかが分かる | リプレイ、見るポイント、ボールマーカー |
| 判断 | 状況に応じた選択の理由が分かる | 采配、守備の読み合い、4th判断 |

## 5. 教材としての基本方針

### 5-1. 説明より体験

先に説明を読ませるのではなく、プレーを選び、結果を見て、必要な場面だけ短く教える。

### 5-2. 必要なときだけ教える

新しいルールや用語は、それが初めて関係する場面で一言だけ登場させる。2回目以降は短く、または表示しない。

例:

- 初めて4thダウンになったとき: 「ここで進めないと攻撃交代。キックで陣地を取る手もあるよ。」
- 初めてファーストダウンしたとき: 「10ヤード進めたから、新しい4回のチャンスだよ。」
- 初めて反則が出たとき: 「黄色いフラッグは反則の合図だよ。」

### 5-3. 詳細学習は任意

リプレイ、用語図鑑、記録一覧、詳しい解説は任意にする。ゲームのテンポを止めない。

### 5-4. 用語はやさしく言い換える

専門用語が出る場合は、マスコットコーチやマネージャーが普段の言葉に言い換えてから使う。

例:

- ファーストダウン: 「10ヤード進めたので、また4回のチャンス」
- パント: 「得点は狙わず、相手を遠くから始めさせるキック」
- インターセプト: 「投げたボールを相手に取られること」

### 5-5. 失敗しても怖くない

ミス、ターンオーバー、反則、タックル、ファンブルは痛々しく見せない。動物キャラがポヨン、コロコロと可愛く動く表現にする。

## 6. ゲーム性との両立方針

教材要素は、ゲーム性の外に置かない。勝つための判断材料として組み込む。

| 教材要素 | ゲーム性への接続 |
|---|---|
| ダウン・残りヤード | どのプレーを選ぶかの判断材料 |
| フィールド位置 | FG、パント、4th勝負の判断材料 |
| 相手の守備傾向 | ラン/パスの読み合い |
| 記録 | 次のプレー判断、学習の振り返り |
| テーピング | 選手状態を見てプレーを変える判断材料 |
| 反則 | ヤードとダウンが変わる状況理解 |
| 用語図鑑 | 現場で聞く言葉をあとから確認する補助 |

## 7. マネージャー視点で特に重要な学習項目

### 7-1. 試合の流れ

- 攻撃側は4回のチャンスで10ヤード進む
- 成功すれば新しい4回がもらえる
- 失敗すれば攻撃権が相手へ移る
- ゴールラインを越えるとタッチダウン
- 4thでは勝負、FG、パントの判断が生まれる

### 7-2. 記録・撮影

- 何ダウンで、何ヤード進んだか
- ランかパスか
- 成功か失敗か
- タッチダウン、FG、ターンオーバーなどの結果
- なぜその結果になったか

### 7-3. 用語理解

- ダウン、ヤード、スナップ、ファーストダウン
- ラン、パス、インターセプト、ファンブル
- FG、パント、キックオフ
- 反則、フラッグ、罰退
- ポジション名

### 7-4. 選手サポート

- テーピングやケアがチームを支えること
- 選手状態がプレー選択に影響すること
- 無理をさせない視点が必要であること

## 8. 新入部員教材としての副次効果

本作はマネージャー向けに作るが、新入部員にも以下の学習効果がある。

- ダウン制の理解
- フィールド位置と得点の理解
- ラン/パス/キックの違い
- 隊形と作戦の狙い
- 守備側の読み合い
- 用語の初歩
- 反則の基本

ただし、新入部員向けに寄せすぎて、マネージャー業務の視点を薄めない。

## 9. 成功条件

教材としての成功条件:

1. 初心者マネージャーが、試合中の「いま何が起きたか」を説明できる
2. ダウン、残りヤード、攻撃権、得点の関係を理解できる
3. ラン、パス、FG、パントの選択理由を大まかに言える
4. 試合記録の基本項目が分かる
5. 用語図鑑を使って現場用語を復習できる
6. 選手ケアやコンディション確認がチーム支援の一部だと分かる
7. ゲームとしてもう一度遊びたいテンポと手触りがある

## 10. 非対象

- 公式ルールを完全網羅すること
- 11人制アメフトの全ポジション・全戦術を最初から再現すること
- 指導者向けの高度な戦術教材にすること
- 長文の講義コンテンツにすること
- 痛みや怪我をリアルに描写すること
