# 設計：アメフトのゲーム性分析（何が面白いか／本作の方向性）

- 作成日: 2026年6月15日
- 目的: 「アメフトとは何か／何が面白いか」を分析し、本作（可愛い・教育主軸・アメフトとして面白い）の設計方針を定める。
- 前提（変えない3条件）: キャラは可愛くてよい／教育用が主軸／アメフトゲームとしての面白さを忘れない。

## 1. アメフトとは何か（本質）

陣地（テリトリー）を奪い合う、ターン制の領土戦。
- フィールドという一本道を、ボールを持って相手エンドゾーンまで運べば得点。
- 攻撃側だけがボールを持ち、守備側は止める。攻守が完全に分かれている（サッカー/バスケと違い流動的でない）。
- 1プレーごとに時計が止まり、毎回ゼロから作戦を組み直す＝「動くチェス」。

## 2. 構造（入れ子の3層）

| 層 | 戦い | プレイヤーの意思決定 |
|---|---|---|
| マクロ | 陣地の戦い（フィールド全体の押し引き） | どこで蹴る/勝負するか |
| ミドル | ダウンの戦い（4回で10ヤード） | 残り距離に応じた作戦選択 |
| ミクロ | 1プレーの戦い（スナップの11対11） | 隊形×プレー、守備の読み |

3層が連動（1プレーの結果→ダウン→陣地→得点）するから、一手一手に意味が生まれる。

## 3. 面白さの正体（3つの快感）

1. 読み合いの快感（マインドゲーム）: 攻撃ラン/パス vs 守備ラン重視/パス重視/ブリッツ。じゃんけん＋情報戦。予測と裏切り。
2. 積み上げの達成感（ドライブ）: コツコツ前進して長いドライブでTD。自分の采配で陣地を削った因果の手応え。
3. 劇的な揺れ（ハイリスク）: ロングTD、ターンオーバー、4th&ゴール。離散的で大きなスイング。

決定的に重要: この面白さは「コーチ（監督）の脳内」でほぼ完結する。選手の身体能力以上に「どの作戦をどの状況で出すか」で勝敗が決まる。だから監督ゲームはアメフトの一番美味しい所を抽出できる＝本作の角度は正しい。

## 4. ゲーム性の核（面白さのエンジン）

football のコアループ:

> 読み合い（選ぶ）→ 実行が見える（起きる）→ 理由が分かる（学ぶ）

決定的な気づき: 教育と面白さは対立しない。同じループの別の深さ。
- 面白い＝相手の裏をかいて作戦が決まる快感。
- 学べる＝なぜ決まったか（ショットガンだから守備ラインを抜けてパスが通った）を理解すること。
- → 「なぜ決まったか」を見せて言葉にすることが、面白さと教育を同時に満たす。

## 5. 現状の問題（実プレイでの観察）

今は「選ぶ→ランダムな結果→次へ」でスロットマシンに近い。
- 結果が「自分の読み」と「相手の手」のかみ合わせで決まる感覚が画面に出ていない。
- 11人が動かず、因果（ラインが空けた→走者が抜けた→タックルされた）が見えない。実プレイでは「キックして誰かが受けるだけ／相手も味方も動かない」に見える。
- → 読み合いも積み上げも劇的さも「見えない」ので、アメフトの面白さが立ち上がらない。
- 一方、采配・守備・キックの「ロジックは正しい土台」（単体/ヘッドレステスト済み）。足りないのは「画面で起こす」こと。

## 6. 面白くする4つの設計原則

1. 結果を「読み合いの帰結」に見せる（スロット脱却）: スナップ前に守備の“気配（テル）”を出す（ボックス密集＝ラン警戒、後ろ厚い＝パス警戒）。読んで裏をかく／読まれて止まる。
2. 1プレーを“振り付け”として見せる（可視化＝教育＋スペクタクル）: 選んだ作戦が11人の連携で起きる（ライン攻防→走/投→守備反応→タックル）。因果が一目で分かる。
3. リスク/リターンを体で感じさせる: ラン=確実小、パス=大きいがリスク、4th/キックの判断。選択に重みを。
4. 積み上げと劇的さを演出: チェーン移動、ゴール接近、観客の動物が沸く。ドライブの手応えとここ一番の緊張を可愛く増幅。

## 7. 教育主軸との両立（マスコットコーチが鍵）

- 毎プレー後、ホイッスル先生が一言で「なぜ」を言う（「密集してたから走るのは止められたね。次はパスが空くよ」）。
- これが「結果の種明かし（面白い）」であり「読み方の教育」。両立点はここ。
- 段階的開放: ラン/パス→守備の読み→4th判断→スペシャルチーム。難易度カーブ（面白さ）＝カリキュラム（教育）。

## 8. 結論と次の一手

本作が作るべき面白さの核 ＝「読み合い → 実行の可視化 → コーチの一言」の1ループ。
これ1つで「可愛い・教育・アメフトとして面白い」が同時に満たされる。

次の一手（優先順）:
1. 1プレーの可視化（全11人が動く：ライン攻防→走/投→守備反応→タックル）＝football の体感を取り戻す最短路。
2. 守備のテル（スナップ前に守備の構えで意図を示す）＝スロットを読み合いに変える。
3. コーチの「なぜ」解説を結果に必ず添える＝教育と種明かしの両立。
4. ドライブ演出（チェーン移動・観客）で積み上げと劇的さを増幅。
